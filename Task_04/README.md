# Task 04 - Controll an RGB LED

## Objective
Make different colors glow on the RGB LED using PWM


## Components Used
- 3 220 ohm resistors
- 1 RGB led
- Jumper wires
- Breadboard 

## Gpio wiring
Cathode (-) --> gnd(pin 6)
Pin 1 (red) --> resistor --> 22
Pin 1 (green) --> resistor --> 27
Pin 1 (blue) --> resistor --> 17
