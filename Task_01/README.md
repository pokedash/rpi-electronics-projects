# Task 01 - Blink an LED

## Objective
Understanding GPIO output by making an LED blink at 1hz

## Components Used
- 1x Red LED
- 1x 200 ohm resistor
- Jumper wires
- Breadboards

## Gpio wiring
LED Anode (+) --> resistor --> GPIO17 
LED Cathode (-) --> GND pin(6)
